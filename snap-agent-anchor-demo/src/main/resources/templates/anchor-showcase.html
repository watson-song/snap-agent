SnapAgent Anchor 功能全览

SnapAgent 锚点功能提供两种模式：Q&A 快问快答和内容注入。 下方展示了所有功能变体，点击各区块右侧的 💬 图标体验 Q&A 模式，注入区域会在页面加载时自动填充内容。

💬 Q&A 模式 ⚡ 注入模式 ⏱️ 缓存控制 👤 千人千面
ℹ️ Q&A 模式：点击区域右侧 💬 图标，打开侧边栏向 AI 提问该区域的内容。
注入模式：页面加载时自动请求后端 skill 生成 HTML 并注入到锚点位置，无需用户操作。
下方每个卡片标注了使用的 data-snap-* 属性组合。

1 Q&A 模式 — 快问快答

用户点击区块右侧 💬 图标，打开侧边栏可针对该区块内容向 AI 提问。支持自动路由、指定技能、关闭技能三种配置。

1.1 自动路由 Q&A（auto skill）

data-snap-anchor="商品分析" data-snap-skill="auto"

2024 Q4 商品分析报告

本季度共管理 SKU 12,847 个，涉及品牌 234 个。其中在售商品占比 78.3%，新上市 SKU 156 个，下市 SKU 43 个。

  • 食品类目增长 12.4%，主要驱动来自休闲零食和健康食品
  • 日用品类目下降 3.2%，需关注库存周转率
  • 电子品类新增 3 个合作品牌，丰富中高端产品线
  • Top 5 热销品牌：清风、维达、良品铺子、三只松鼠、小米

建议：重点关注食品类目的补货策略，日用品需启动清库促销。

1.2 指定技能 Q&A（specific skill）

data-snap-anchor="库存预警" data-snap-skill="trend-prediction"

库存预警面板

以下 SKU 库存水位低于安全阈值，建议尽快补货：

  • SKU-A001 清风抽纸 200抽：当前库存 45箱，安全库存 200箱 ⚠️
  • SKU-B023 良品铺子坚果礼盒：当前库存 12箱，安全库存 80箱 ⚠️
  • SKU-C107 小米 chargar 20000mAh：当前库存 8个，安全库存 50个 🔴

已启用指定技能 trend-prediction 进行趋势预测问答。点击 💬 提问库存趋势。

1.3 关闭技能 Q&A（skill off — 纯上下文对话）

data-snap-anchor="操作日志" data-snap-skill="off"

今日操作日志

  • 09:12 admin 创建了补货计划 PO-2024-1206
  • 10:34 zhang_san 审核通过调拨单 DB-2024-0398
  • 11:08 li_si 修改了 SKU-A001 的安全库存为 200
  • 14:22 admin 导入了 45 个新 SKU
  • 15:30 system 自动触发清库预警

技能已关闭 (skill="off")，AI 将仅基于本区域内容回答，不调用任何技能工具。

1.4 自动发现锚点（无需 data-snap-anchor）

section id="auto-section" （自动识别）

自动发现区域

当页面中没有显式标记 data-snap-anchor 的元素时，SnapAgent 会自动扫描 <section>、<h2[id]>、<h3[id]> 元素并为其添加锚点。

本区域使用了 id="auto-section"，没有 data-snap-anchor 属性，但 💬 图标仍然会出现。

2 注入模式 — 自动内容生成

页面加载时自动请求后端 skill/workflow 生成 HTML 并注入锚点位置。支持缓存控制、千人千面、fallback 降级、workflow 调用等。

2.1 Skill 注入 + 1小时缓存 + Fallback

data-snap-mode="inject" data-snap-skill="announcement" data-snap-cache-ttl="3600" data-snap-fallback="..."

2.2 个性化欢迎卡片（千人千面）

data-snap-mode="inject" data-snap-skill="welcome-card" data-snap-cache-ttl="1800"

2.3 每日提示 + 24小时缓存

data-snap-mode="inject" data-snap-skill="daily-tips" data-snap-cache-ttl="86400"

2.4 天气小部件

data-snap-mode="inject" data-snap-skill="weather-widget" data-snap-cache-ttl="7200" data-snap-fallback="..."

2.5 实时推荐（不缓存，每次刷新都不同）

data-snap-mode="inject" data-snap-skill="realtime-recommend" data-snap-cache-ttl="0" data-snap-fallback="..."

2.6 Workflow 注入（工作流驱动）

data-snap-mode="inject" data-snap-workflow="daily-summary" data-snap-cache-ttl="3600"

3 属性参考

以下是 SnapAgent 锚点支持的所有 data 属性及其用途。

属性 模式 说明
data-snap-anchor通用锚点名称，用于上下文标识
data-snap-mode注入设为 "inject" 启用注入模式，不设则默认 Q&A
data-snap-skill通用技能 ID，Q&A 模式可用 "auto"/"off"/技能名，注入模式指定 skill
data-snap-workflow注入工作流 ID，与 skill 二选一（skill 优先）
data-snap-cache-ttl注入缓存 TTL（秒），0 = 不缓存，默认 3600
data-snap-fallback注入注入失败时的降级 HTML 内容
