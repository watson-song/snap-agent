SnapAgent v0.4 Anchor Feature Demo

SnapAgent 锚点功能演示

SnapAgent 锚点为 Web 页面提供 AI 驱动的内容交互能力。 支持页面快问快答 (Q&A) 和自动内容注入 (Inject) 两种模式， 实现千人千面的个性化页面体验。

💬

Q&A 模式演示

点击区块右侧 💬 图标，打开侧边栏向 AI 提问该区域内容。支持自动路由、指定技能、关闭技能、自动发现。

data-snap-anchor data-snap-skill
⚡

注入模式演示

页面加载时自动请求后端 skill 生成 HTML 并注入。支持缓存控制、降级 fallback、workflow 调用、千人千面。

data-snap-mode="inject" data-snap-cache-ttl
