SnapAgent 是一个嵌入式诊断 Agent 库。Spring Boot 2.x 应用引入一行 Maven 依赖，即可获得：自然语言诊断对话、只读工具调用（JDBC / Redis / 代码 / 日志 / 指标 / 链路）、业务知识自动注入、代码图谱、多步工作流、问题闭环与主动巡检。 SnapAgent is an embeddable diagnostic Agent library. Add one Maven dependency to your Spring Boot 2.x app and gain: natural-language diagnostic chat, read-only tool calls (JDBC / Redis / code / logs / metrics / traces), auto-injected business knowledge, code graph, multi-step workflows, issue closure loops, and proactive patrols.

Core 模块只定义 SPI，Starter 模块提供 Spring Boot 自动装配与内置实现。宿主可以替换任何一个 SPI bean。 Core module defines only SPIs; Starter module provides Spring Boot auto-configuration and built-in implementations. Hosts can replace any SPI bean.

快速开始Quick Start

两步集成，立即诊断 Two steps to integrate, diagnose instantly

在你的 Spring Boot 2.x 应用 pom.xml 中加入依赖，配置 LLM 凭证即可。 Add the dependency to your Spring Boot 2.x app's pom.xml, configure LLM credentials, done.

<!-- 1. 添加 SnapAgent starter 依赖（含 core） --> <dependency> <groupId>cn.watsontech.snapagent</groupId> <artifactId>snap-agent-spring-boot-2x-starter</artifactId> <version>1.0-SNAPSHOT</version> </dependency> <!-- 2. (可选) REST API SDK 用于外部调用 --> <dependency> <groupId>cn.watsontech.snapagent</groupId> <artifactId>snap-agent-client</artifactId> <version>1.0-SNAPSHOT</version> </dependency>
# application.yml — 最小可跑配置 snap-agent: enabled: true base-path: /snap-agent # web UI + REST API 挂载路径 llm: api-type: anthropic # 或 openai (通义/文心/智谱) base-url: https://api.anthropic.com auth-token: ${ANTHROPIC_API_KEY} model: claude-sonnet-4-6 timeout-seconds: 60 security: required-permission: snapagent:use # 复用宿主 Spring Security # 可选功能模块（默认 disabled）——完整字段参考： # https://github.com/watson-song/snap-agent/blob/main/docs/site/integration/zh/host-integration-guide.md code: # 代码工具（code_read / project_structure / code_graph） enabled: true project-root: ${user.dir} knowledge: # 业务知识库（自动注入到 LLM system prompt） enabled: true sources: - type: markdown dir: classpath:/docs/knowledge/ # cost / workflow / issue / patrol 同理开启即可，详见上方链接
# 启动你的 Spring Boot 应用，打开浏览器访问： # http://localhost:8080/snap-agent/ (使用宿主应用的登录凭证) # 或用 REST API 直接调用： curl -u demo:demo -X POST http://localhost:8080/snap-agent/runs \ -H "Content-Type: application/json" \ -d '{"skillName":"health-check","inputs":{"time_range":"1h"}}' # 响应： { "taskId": "task-abc123", "status": "running" } # 订阅 SSE 流式输出： curl -N -u demo:demo http://localhost:8080/snap-agent/runs/task-abc123/stream
